# Notice
- Use Super+Shift+C to Launch the color picker. (can be set in schemas/org.gnome.shell.extensions.color-picker@gnome.gschema.xml)
- Tested on Gnome 49.3 and gnome 50.0

# Dependencies
- Python

# Installation
- Clone and then copy to the gnome-shell extensions path:
- 1- git clone https://github.com/MAKalantari/Gnome-Color-Picker-Extension.git
- 2- cp ./Gnome-Color-Picker-Extension ~/.local/share/gnome-shell/extensions/
- then logout and log back in.
